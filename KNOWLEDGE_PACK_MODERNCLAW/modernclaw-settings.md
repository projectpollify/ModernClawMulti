# ModernClaw Settings

## Purpose
Settings controls appearance, model defaults, behavior toggles, privacy, storage visibility, and the voice system.

## Where To Find It
Open the `Settings` view from the sidebar or the header settings button.

## Main Settings Areas
- Appearance
- Model
- Behavior
- Voice Output
- Voice Input
- Privacy
- Storage
- onboarding restart and reset actions

## Important Settings
### Appearance
- theme selection

### Model
- default model
- context window size

### Behavior
- stream responses
- send on Enter
- show token count

### Privacy
- save conversation history

### Voice Output
- enable voice output
- approved voice selection
- Piper executable path
- Piper voice model path
- refresh status
- test voice

### Voice Input
- enable voice input
- Whisper executable path
- Whisper model path
- language
- refresh status

### Storage
- memory folder display
- open memory folder
- reset settings
- restart onboarding

## Important Product Rule
Settings is where the user controls the system contract. It is not just cosmetic configuration. Changes here can directly affect prompt context size, persistence behavior, active model, and voice readiness.
