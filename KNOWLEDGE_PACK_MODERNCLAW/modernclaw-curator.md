# ModernClaw Curator

## Purpose
The Curator Inbox is the staging layer between raw external research and live knowledge.

## Where To Find It
Open `Brain` and look for the `Curator Inbox` section.

## Current Multi-Brain Truth
- Curator is scoped to the active brain
- importing a package affects only the active brain
- switching brains should not show another brain's staged or approved curator content

## Main Actions
- refresh staged packages
- import a staged package into live knowledge
- reject a staged package

## Important Notes
- Curator remains a human-reviewed staging model
- it is not autonomous live mutation
